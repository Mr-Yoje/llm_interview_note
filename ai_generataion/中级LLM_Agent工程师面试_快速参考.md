# 🚀 中级LLM/Agent工程师面试 - 快速参考

## 🔑 **核心知识点**

### 1. Transformer架构（必问）
- **Q**: 解释Multi-head Attention的工作原理
- **A**: 多头机制允许模型在不同位置关注不同子空间信息，通过并行计算多个注意力头然后concatenate

### 2. 推理优化（高频）
- **Q**: KV Cache如何工作？有什么优化策略？
- **A**: 缓存Key和Value向量避免重复计算，优化包括内存池、分页管理(PagedAttention)、量化

### 3. Agent系统设计（重点）
- **Q**: 设计一个多Agent协作系统
- **A**: 包含任务分解、角色分配、通信协议、状态同步、错误处理机制

### 4. RAG系统（实用）
- **Q**: 如何优化RAG系统的检索质量？
- **A**: 智能分块、多向量检索、重排序、查询扩展、上下文感知

## ⚡ **编码题模板**

### PyTorch优化
```python
# 实现高效的Top-k采样
def top_k_sampling(logits, k):
    values, indices = torch.topk(logits, k)
    probs = torch.softmax(values, dim=-1)
    choice_idx = torch.multinomial(probs, 1)
    return indices[choice_idx]
```
